Numeric Polygon Puzzle Solver

The objective of this puzzle is to arrange the n given polygons so that the numbers on each piece is positioned next to the same number on the adjacent pieces they touch. This was solved by using multiple pruning methods to exponentially decrease the number of possible permutations.
